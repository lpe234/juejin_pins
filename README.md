# juejin_pins

掘金沸点简单分析

```text
├── article  # selenium示例
│   ├── __init__.py
│   ├── file
│   ├── get_article.py
│   └── parse_article.py
├── pins  # 获取沸点示例
```

## 目录结构

### article

Selenium + Chrome 获取网页数据。

### pins

`get_pins.py`获取沸点数据示例。

`get_pins2.py` 获取全部沸点数据。
